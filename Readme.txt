git clone 
地址复制到后面

git add .
将改动的地方添加到版本管理器

git commit -m "changes log"
提交到本地的版本控制库里，
引号里面是你对本次提交的说明信息。

git push -u origin master  
将你本地的仓库提交到你的github账号里，
此时会要求你输入你的github的账号和密码。N9d --> G-v
